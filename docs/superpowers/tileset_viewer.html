OU Game Tileset Index Viewer

Open this tool to find index values for editing your map.map file.
How to use for editing map.map:
  • Every tile is annotated with its 2-digit index code (from 00 to 10 for rows/cols).
  • Row index represents tens digit, Column index represents units digit. E.g., Row 3 Column 3 is 33.
  • Use these 2-digit values in the first block (Ground) or second block (Decor) of your map.map file.
